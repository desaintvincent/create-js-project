## Install
```sh
yarn install
```
## Run
| command       | description           |
|---------------|-----------------------|
| yarn dev      | run in dev mode       |
| yarn build    | build the application |
| yarn lint:js  | lint all js files     |
| yarn lint:css | lint all css files    |
| yarn test     | run unit tests        |
